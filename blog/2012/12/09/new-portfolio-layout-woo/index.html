New Portfolio Layout, Woo~

Dec 9th, 2012


Hurray!

I have to say it’s a pretty huge improvement over the old layout I’ve had for over a year now, and it’s due to the fact that I can do a ton of front-end work at my new job. I can’t believe how much I’ve learned over the past four months, including but not limited to standard stuff like jQuery that I have always wanted to practice but hadn’t found the time for.

This particular layout makes use of a new CSS trick that I learned, and really like – a way to resize the “top” div of a website so that it always fits in the space above the fold, no matter the size. All you need is a container div, and inside that two more divs: one for the area above the fold (the “landing”), and one for your “main” information.

The CSS looks like:

html,body {
margin:0px auto;
height:100%;
}

div#container {
min-width:100%;
height:100%;
}

div#landing {
width:100%;
height:100%;
background: #5d5d5d url("../images/header_bg.jpg") center center no-repeat;
background-size:cover;
}

div#main {
background-color:#eeeee2;
}

And the HTML should be like:
<body>
 
    top part of page
  
 
    the rest of the page
  
</body>

How exciting.
