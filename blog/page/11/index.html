Some HTML5/JavaScript Links

Aug 16th, 2012

Having done few HTML5 things, I’ve dug up some tutorials and interesting links.


HTML5 canvas - the basics

I guess the first cool thing about HTML5 is this canvas thing, which is basically like a drawing pad that HTML5 bestows upon you, eyes glowing with love. “Go,” it says. “Use this to create art with your present armory of JavaScript.”

A canvas object has a bunch of contexts, the same way a drawing pad has a lot of pages. JavaScript can be used to draw onto a context. Most browsers allow JavaScript to draw 2D things; some have experimental 3D contexts.

Unlike a drawing pad, once you’ve decided to use a canvas’s context, that’s the only context you can use for that canvas, ever. It’s kind of like you tore out a page from the drawing pad and all the other ones withered away in anguish.

Interesting things canvases can do in HTML5:
  • draw lines, ex. canvas.lineTo(xPos, yPos)
  • draw rectangles, ex. canvas.fillRect(xPos,yPos,width,height)
  • draw images, ex. canvas.drawImage(imgSource, xPos,yPos)
  • apply pixel manipulations to those images, ex. invert all colors
  • draw text (only in recent WebKit builds)
  • draw dropshadows and gradients for all your Web 2.0 needs


Understanding save() and restore() for the Canvas Context

When searching around for examples of canvas usage, I saw references to canvas.save() and canvas.restore() which were nicely explained in this article.

In brief, every canvas contains a stack of “drawing states.” Drawing states are comprised of a lot of data, including the current transformation region, the current width of line being drawn, the current front, and so on.

If you want to draw a shape like a diamond (or rotated square), the steps to do this involve:
  1. Make a canvas, retrieve its 2D context
  2. 2Dcontext.save()
  3. Rotate the context
  4. Draw a square on the context (which, since the canvas is rotated, will be diamond-shaped)
If you then wanted to draw a new shape, like a regular square, you could easily pop the present drawing state with 2Dcontext.restore() to get back to an un-rotated coordinate system. The diamond should stay the way that it is presently.

Similarly, objects on a canvas can’t be “just moved;” if you’re translating an object across the canvas, the steps go
  1. Clear the canvas
  2. Draw the object in its new position
  3. Repeat

JavaScript function declarations vs. function expressions

In the previous tutorial, I encountered my first instance of a function expression. Essentially, a function definition looks like
 function function_name() {
   // stuff
  }

whereas a function expression looks like
 var = function function_name() {
   // stuff
  };

Some links I found which discussing declarations versus expressions are Function Declarations vs. Function Expressions and Named function expressions demystified, and the differences include the following:

  • Function expressions can be named or anonymous.
  • Function declarations are parsed and evaluated before any code is executed, whereas function expressions are evaluated at run-time. Therefore, if you call a declared function at any time, it should be okay, whereas function expressions must be defined before being called.
  • Function declarations should not be used in if-statements. In contrast, function expressions are okay to call in if-statements, and are sometimes used this way to allow for best performance (“don’t define this function unless it’s needed”).
  • Function expressions mimic what JavaScript is actually doing, which is creating an object every time a function is defined.

In general it seems that function expressions are the favored creature.

Objectified

Aug 13th, 2012


Objectified is a feature-length documentary film examining the role of everyday non-living objects, and the people who design them, in our daily lives. The film is directed by Gary Hustwit. Objectified premiered at the South By Southwest Festival on March 14, 2009. (via Wikipeda)

Objectified has been languishing on my Netflix Instant Queue for the good part of a year or so. I finally watched it, and was glad! The film brings together a variety of designers from Across the World, who all talk about what they personally feel is Good Design. It’s refreshing to watch designers talk so passionately about chairs and personal marks on the world and shaving away at the superfluous “fat” of a toothbrush design until they eliminate the concept of a toothbrush from oral health completely. It’s amusing to see one designer swear that he’s always “lived in the future,” while another reaches into the past for a modern cellphone design inspired by childhood memories of peeling potatoes.

In the end, I was relieved when the movie moved from “NEW THINGS! Aren’t they cool? Buy all these new cool things!” to “Do we really need new cool things? I mean, just look at all the things that we already have.”

just…look at them (source)

The film points out the great problem of sustainability in product design, how it’s not just about recycling/upcycling but how sustainability must guide design from before the beginning to beyond the end. Why are you designing this thing? What will it add to the world that doesn’t already exist? And what will happen to the thing, when someone designs a new thing five seconds later?

Visual Merchandising, Tony Morgan

Aug 8th, 2012

cover of visual merchandising (source)
A great introduction for retail students, this book offers a user-friendly reference guide to all aspects of visual merchandising and covers both window dressing and in-store areas. Using examples from a range of stores from fashion emporia to supermarkets, the book offers practical advice on the subject, supported by hints and tips from established visual merchandisers. It reveals the secrets of their tool kit, and information on the use of mannequins, the latest technology, how to construct and source props, and explains the psychology behind shopping and buyer behavior.

Presented through color photographs, diagrams of floor layouts, and store case studies, and including invaluable information such as a glossary of terms used in the industry, Visual Merchandising is an essential handbook for anyone working in and learning about this exciting area.

This came up as a hit when I searched the public library database for books about stores and merchandising, which I was in turn searching for as a resource for free-to-play game design. As NPR uncovered, free-to-play games are a lot like stores; you see the nice displays outside, shuffle in and around for a while, and if you discover something that you like, something that will, perhaps, make you feel like a better human being, you pay money to obtain that thing.

Electric Rest details some other possible applications of store design to free-to-play games, and I thought I would try and find some myself. Visual Merchandising turns out to be something of a beautiful textbook, with a history of window displays that in Olden Times (a.k.a. early 1900s) displayed not fashionable manikins, but rotating displays of art and sometimes curiosities (like a plane that had fallen while attempting to cross the Atlantic, and was promptly reassembled in a department store window).

With regards to a digital medium, the takeaways are a little harder to come by, though I guess you could force them out. For example, indoor arrangements depend on the store’s architecture, but there are a couple rules of thumb that help, such as “shelves that are angled rather than parallel to each other can draw people in to explore.” Could an interface be designed in such a way that the eye is drawn to viewing other goods for sale, rather than relying on a “Go to next page” click? How could you fashion an experience that allows someone to wander aimlessly, yet also not impede them if they want to get directly to Point B?

The three-dimensional space and atmosphere afforded by a store can be impactful in ways that I think are hard to mirror to video games, or even shopping websites. There’s the three-dimensionality, of course, and the impact of sound and other people and smells set to artificially make you feel like you’re in a warm tropical clothing paradise ripe for bikini buying. Also, even though I’m sure people are aware of the fact that stores try and push cheap gum and parcels of assorted high fructose corn syrup at you as you’re paying for your Actual Purchases, I think it’s easy to write off a lot of store design as “Oh, they did that just because that’s the way the store is shaped, not because they’re trying desperately to sink their hands into my wallet.” Contrast to perpetual social game pop-ups rewarding you with discounts on virtual goods.

Another thing that I believe stores will have going for them for a while is their ability to support social exploration, to serve as a bauble-filled backdrop to people’s wanderings as they share each other’s company. I can’t think of any games that do similar. (Though I suddenly imagine some WoW- or Second Life-esque scenario…ah, well, you could imagine an MMORPG fitting into any scenario, though.)

In short: Visual Merchandising has gorgeous photos, some interesting history and details about things like department stores and manikins, and so on. It also features some ~*famous window displays*~ like the one from Selfridge’s which has all those inflatable boobs…what does it look like? Read the book!

The Fault in Our Stars, John Green

Aug 8th, 2012

(source)

Diagnosed with Stage IV thyroid cancer at 13, Hazel was prepared to die until, at 14, a medical miracle shrunk the tumours in her lungs… for now.

Two years post-miracle, sixteen-year-old Hazel is post-everything else, too; post-high school, post-friends and post-normalcy. And even though she could live for a long time (whatever that means), Hazel lives tethered to an oxygen tank, the tumours tenuously kept at bay with a constant chemical assault.

Enter Augustus Waters. A match made at cancer kid support group, Augustus is gorgeous, in remission, and shockingly to her, interested in Hazel. Being with Augustus is both an unexpected destination and a long-needed journey, pushing Hazel to re-examine how sickness and health, life and death, will define her and the legacy that everyone leaves behind.

I finished this book in an evening and a morning. It is fantastic. A review on this ain’t livin sums it up more nicely than I feel I could: ”The Fault In Our Stars basically took every cancer kid trope and shat on it, hilariously and sensitively and brilliantly.”

s.e. smith continues:
The Fault In Our Stars is a book that makes cancer funny, that pokes fun at cancer, that points out the absurdity of life and shows you that it’s okay to be, and feel, ridiculous while dealing with chronic illness at the same time that it can also be funny, and frustrating, and enraging, and depressing. It’s a far cry from the usual sombre texts about cancer or the media that suggest cancer patients need to crack jokes for the benefit of the people around them. The dark humour is about insider’s jokes: It’s for people with cancer, not for others. It’s funny if you’ve been chronically ill, and isn’t intended to amuse caregivers and family and friends, although some may well find it funny too.

It’s also a book that unflinchingly faces tragedy without turning maudlin and ridiculously sentimental. The Fault In Our Stars points out that the facts of life are often really, really crappy. For a man dedicated to reducing world suck levels, I like that Green pointed out that sometimes, world suck levels are out of our control, and that some of the narratives for dealing with that are deeply unhelpful and unuseful.
Sometimes the characters’ dialogue reminded me of something out of Juno, a little long and strewn with witticisms, but this stuff struck me less as “Man this is so implausible :(” and more as “This is so much fun to read, why am I suddenly laughing and crying :):”

Another (weird, flinchy) plus for me: Hazel has two friends that are guys. And yet. A love triangle does not happen.

What else can I say about it? Things happened. Characters did stuff. Most importantly, everything that happened had weight. Go read it already!

Lone Survivor

Aug 7th, 2012


I’ve finally gotten around to playing some games I’ve obtained over the past couple months, including Lone Survivor, which is (thus far) set in a post-apocalyptic apartment building with questionable lighting. Said building brims with epidemic-stricken people-turned-zombie-things and a variety of scenarios that invite you to question whether any of this is Actually Happening.

Because I’m a “min-max” type of player, games like this can do a number on me; I hate wasting in-game time/flares/bullets/food rushing down zombie-filled corridors that yield only dead ends, and upon finding myself in this situation I usually beat a hasty ESC and restart the game. Survival horror’s tendency to translate its protagonist’s physical/mental health as a reddening vignette or extra panting animations bewilders me. I want to be 100% all the time, damn it! Tell me exactly when I need 10 more points of food to be fully sane!

The map is difficult to parse at first, but it expresses my favorite little detail; in darkness it remains in shadow, so that to read it easily you have to be under actual sources of light. (Whoa! So realistic!)

Two Wishes for Orca Card

Jul 30th, 2012

photo by JeanineAnderson (source)


I’ve written before about the Orca Card, which is essentially Seattle’s digital purse for public transportation: buses, ferries, our single light-rail system. Now that it’s summer and I’ve come to rely more on my Orca Card instead of my beloved (yet inactive) student bus pass, a couple things become apparent:

  1. I really wish I knew when my transfers ended. I don’t even remember how long transfers are supposed to last. Two hours? Three? When you pay in cash, the bus driver hands you a little piece of paper which can be used as fare for your next bus ride, as long as you’re fairly within time limits. With Orca, your transfer is invisible, and strict – are you leaving the bus two minutes after your fare is up, because of traffic or because the bus itself is late? Too bad, you’re paying the fare again. (This also gets frustrating when you board a bus that’s “Pay as you leave” – because even if you got on the bus within the time limit, what if you disembark too late?)
  2. I really wish I knew how much money I had on my card as soon as I looked at it. In my mind I imagine a little E-ink box that would happily display your available funds, in a futuristic, perfect society. To be fair, the card reader does tell you what your card’s remaining balance is once you swipe, but I find myself squinting at the small, gray-on-gray display every time I pass by. And, it would also be nice if this card reader could tell you things like, “Hey, your transfer will be up at 4:15pm!”

Not knowing when your transfers end can also be problematic for low-income users; as a professor in the HCDE department at UW points out from a student dissertation, “Basically, you board the bus and swipe your card. If you’re inside the allotted transit time, nothing is deducted from your card. If not, the full fare comes off your card. And for some people, that might have been their way to work tomorrow. And that’s it for their cash this week.”

A Guide to Some Touristy Things of O’ahu

Jul 27th, 2012

I spent the week of the Fourth of July visiting Hawai’i for the first time. We went to the third-largest island, O’ahu, a.k.a “The Gathering Place,” which is the island which has Hawai’i’s capital, Honolulu. Our driver from the airport described Honolulu (“place of shelter”) as being named for the fact that the surrounding mountains shield it from bad weather.

o’ahu (source)

SpeediShuttle, which has a partnership with the airport, did a great job of taking us to the Seaside Hotel with a minimum of stress despite parades clotting up downtown Honolulu. We ended up paying around $50 for a round-trip, with the promise that we could call their 24-hour hotline at any time and get picked up and taken back to the airport when we needed to catch our departing flight. I wouldn’t recommend paying in advance for the returning trip to the airport, because no one came to our 5am pickup or picked up the phone, and an hour and a half before our departure time we resorted to taking a cab back to HNL, which cost an extra $40. :( Edit: A SpeediShuttle update.

We stayed at the Seaside Hotel, which is one of the more modest places in Waikiki, but still well within reach of buslines and tourist fervor. Some online reviews indicated that it used to be “private” and reserved for airline employees. You can rent beach towels for the day, swim on a small rooftop pool, and lie on the bed wondering in vain why Discovery Channel broadcasts so many “[Storage/Property/X] Wars” shows nowadays. There was free Wi-Fi but we were never able to really connect to it successfully, and relied on the lobby computers for bus schedules instead. (To be honest, it was nice to be on a forced Internet break.)

Something that our SpeediShuttle driver recommended we check out was a magazine called 101 Things to Do: Oahu, which turned out to be a pretty good guide-slash-advertising booklet. This and other magazines can be picked up for free on most street corners, and contain coupons!

101 Things to Do: Oahu (source)

Monday
The first thing we did was go to Hanauma (“ha-now-ma”) Bay Nature Preserve, which is essentially a really gorgeous bay historically used by Hawaiian royalty as their private fishing grounds.

iphone photos stitched together with hugin

Admission fee is $7.50. The coolest thing to do here is snorkel; the water is warm and colorful fish dart around pecking at the rocks. (Apparently turtles, sharks, and jellyfish sometimes drift through as well; and we definitely saw at least one pufferfish that had us flailing back to the shore in noob-like panic, haha.) You can rent snorkeling sets (goggles, snorkel breather thing, enormous awkward fins) for $15, or $18 if you want prescription goggles (which were worth it).

Other wildlife to be spotted include Indian Mongooses, one of which I spotted sneaking around beach towels placed too close by shady foliage. Mongooses are invasive, and were introduced in the 1700s to the islands to deal with rats eating up the sugarcane. (In the end, diurnal mongooses ended up snacking not on noctural rats, but on the eggs of endemic bird species. Whoops.)

Hanauma is open all days except its Tuesday sabbath, and is accessible via the 22, which takes 40 minutes to an hour to get there from Waikiki. Because of the crowds, it’s best to get there early and leave early, since afternoon buses come every half-hour and sometimes can’t pick everyone up. (The 22 fills up with visitors from the Sea Life Park before accepting people from Hanauma.)

The restaurants we visited that day were Lulu’s for breakfast, and Marukame Udon for lunch. Lulu’s had great loco moco and macadamia coconut waffles (though the latter were quite syrup-drenched and sweet). Marukame ended up being our favorite restaurant the whole trip: the udon is freshly-made, cheap (at about $6 a bowl in Waikiki, where prices are tourist-high), and delicious. The tempura isn’t bad either. :9

The best times to go to Marukame are around 3 to 4; after that, it’s dinner rush.

Another place we visited throughout our trip was Tropical Dream Ice Cream, a hole-in-the-wall ice cream place right next to the Seaside Hotel. It served delicious ice cream with local ingredients, including but not limited to: mango, pineapple coconut sorbet, macadamia nut…

macadamia nut tastes great with john steinbeck


Tuesday
We spent this day walking through downtown Honolulu to try loco moco at L&L’s, which is kind of like a Hawaiian McDonalds, except all the food comes on white styrofoam trays. (It was pretty good, though almost overly-filling. Then again, maybe we hadn’t recovered from Marukame the previous day.)

The main goal for the day was to visit the Foster Botanical Garden, one of the many botanical gardens on O’ahu. The guides at front said it would take about an hour to get through. We went a little fast, but still enjoyed the enormous “exceptional” trees, which included this absolutely enormous palm tree:

you are so big!

Another cool tree is thorny, percussive cannonball tree (not pictured), and the striped rainbow eucalyptus (below).

you are so colorful!

We also visited the Waikiki Aquarium, which is fairly small but worth a visit if neon-colored fish are your thing. (The giftshop at Hanauma also offers a coupon for $1 off adult admission to the Waikiki Aquarium.) The most impressive part to me was the “enormous giant huge fish exhibit,” in which various person-sized fish like the giant trevally circle around.

you are so, so big

cue the children screaming “NEMOO”

I think we also tried hiking up to Makapu’u on this day. It’s reachable by taking the 22 or 23 to Sea Life Park and walking back and up along the highway. The trail is simple – just an upward curve – and at the top is a lookout point with a small lighthouse.

the breeze was so strong and nice

I liked the view back down almost better.

i can’t remember the name of this mountain, though. diamondhead?


Wednesday
On Wednesday we headed out super early to visit Eggs n Things, a restaurant in Waikiki where we had seen a pretty sizable line of people waiting at around 7am. Luckily, time zones made it easy for us to wake up and head out in time to avoid a 20+ minute wait, and the food (loco moco, and banana/macadamia nut french toast) turned out to be just as delicious as hoped. Eggs n Things also has their own triplet of syrups, flavored guava, pineapple, and coconut. They were so delicious I bought the coconut one to take home, only realizing later and with sorrow that one ingredient is partially hydrogenated oil.

loco moco & fancy french toast at eggs n things

It was the first Wednesday of the month, so we were intending to visit the Honolulu Museum of Art on their free day, but because it was also a holiday (July 4th), the museum was closed. We headed up to the Bishop Museum instead, which was great and had a variety of Hawaii-specific exhibits.

bishop museum courtyard

A science building describes Hawaii’s biological diversity and formation with wax volcano simulations, and other buildings contain many cultural and historical artifacts from the Asian immigrants who worked on the sugarcane plantations. We were also able to catch a show in the observatory that showed us where and how to spot Mercury and Venus in the night sky.


Thursday
Thursday was our dedicated “North Shore” day. We caught early buses to Waimea Valley Adventure Park. It took two hours to get there, but the drive was quite scenic.

a beach near/in waimea valley

Waimea stands for “red water,” and I think refers to how the dirt is reddish with oxidation? (Citation needed.) Waimea Valley Adventure Park itself is a huge botanical garden which leads up to a freshwater waterfall where visitors can swim if rainfall hasn’t made the area mudslide-hazardous.

waimea falls

The falls are cold and 30 feet deep in areas, and floats are provided for people who aren’t strong swimmers.


Friday
Our last day in Hawaii was spent visiting Kailua. Kailua has the whitest, finest sand on O’ahu, but is also littered with little pinecones and needles. The day we went, strong wind turned the sand on the shore into fine little bullets, though underneath the waves it was velvet-soft. You can kayak here to a nearby rocky-looking island, but seeing as I can’t get 20 feet from the shore without fearing I’ll be swept into the Pacific Ocean, I was content watching people flail their oars around against the waves.

Restaurant recommendation: Boots n Kimos, cash-only, which has very yummy pancakes drenched in macadamia nut cream.


In Conclusion
O’ahu! It was great.

Lavender Festival & Hurricane Ridge

Jul 26th, 2012

purple haze farms

Last weekend I visited the Sequim Lavender Festival. Fun fact: Sequim is the only city in Western Washington that has an irrigation system. (At least, that’s what the volunteer on our bus to Purple Haze Farms said. Even if it’s not quite the only irrigation system presently, I guess the idea is that Sequim doesn’t get as much water as surrounding areas.)

chicken statue and lavender

The Lavender Festival is fun and fragrant. You can tour farms and a broad and variety-filled street fair. The population of European-descended Americans is such that a lot of stuff plays on fairy myths (bottles of “fairy dust,” beaded “fairy bells,” hose-and-wire “fairy wings”), which I rarely see at other street fairs I attend in Seattle. Fun fact: lavender is part of the mint family, has antiseptic and anti-inflammatory properties, and the price of a pound of it in Roman times was equivalent the price of fifty haircuts.

When I approached this next plant at Purple Haze, someone declared, “Teazle has been around since the time of the woolly mammoth!” Wikipedia doesn’t back this up, but does describe teazle as invasive in the United States and partially carnivorous.

teazle/teasel/teazel, aka dipsacus

My proudest purchase was half a dozen sedums/phocuses/succulents/stonecrops/hens-n-chicks of various colors, whose specific species I can’t recall or discover. (Maybe one’s a Sedum album rhodopaeum?) Fun fact: plants like this must remain outdoors, because their stoma, which is used in photosynthesis and respiration, use a drop in temperature as a signal to open up. So, sedums in an indoor environment with relatively stable temperature suffocate.)

i can breathe!

My collection of relatively hard-to-kill plants (started with a pair of air plants I got at another street fair) is growing! (Haha?)


Afterward we visited Hurricane Ridge. It costs $15 for a car to enter, but you have in-and-out privileges for about a week and can ascend some 5000 feet up and over the clouds.

whoa!

Fun fact: Hurricane Ridge is named for its “intense gales and winds,” though all we had when we visited was sunshine and slight elevation sickness.

Racermate Wireframes

Jul 23rd, 2012

“before and after” comparisons for racermate wireframes

A couple days ago I made up wireframes for a theoretical redesign of the Racermate website, where a couple friends of mine work.


Front Page

First was the front page, which I found personally bewildering: the buttons that frame either side of the wall of “newsletter” text at the center of the page don’t have much context, or direction. “MultiRider Software,” “MultiRider Center Locations” – what is that, what do I do with it, why is it relevant? “Wingate Testing with Velotron” – what does this indicate or mean to a new user? “ErgVideo.com” seems to open a new website completely, how is it affiliated to Racermate, what does it do?

For the redesign, I focused on making the site informative to new users. I kept much of the information in the header “The WORLD LEADER in ELECTRONIC BICYCLE ERGOMETERS…” and “OFFICIAL BIKE TRAINERS OF…” – because they’re good explanations of “This is who we are, this is why we are trustworthy.”

I made the top navigation buttons larger than in the original design, and made them more self-explanatory: “COACHES” in the original turned into “Find a coach;” “MultiRider” became “MultiRider: Ride Together” to give the user more grounding on what they would learn about when clicking through. This would also expand users’ knowledge of Racermate products right on the front page; MultiRider can be immediately understood as a social application of a RacerMate product.

The next two “rows” of information also serve to focus and explain Racermate through images, rather than text. The first row is a carousel of images, text description, and links to different Racermate products; the next, a much more involved “This is who we are” statement.

Because word of mouth is important to this product, part of the front page is also devoted to testimonials sent in by happy consumers.


Product Display Page

Talking with friends at Racermate, I learned that online orders make up a little more than half of the total number of orders that this company gets. But when I went through the website myself, I found the order page bewildering and cluttered with a variety of colors of clots of text. As a new user unfamiliar with the product, I didn’t understand what exactly the product would be when I ordered it, or what would come with it, or what exactly it is meant to do. To get a full picture of what some products do requires that the user navigate to half a dozen completely different pages. Ordering information is also separated from a product overview, and ordering online requires that the user:

  1. Learn about the product by navigating around “Products” and all related pages in the top navigation
  2. Click “Sales” in the top navigation
  3. Hover over desired product
  4. Click “Pricing & Information”
  5. Parse a long order page
  6. Click “Add to Cart”
  7. Complete order

For the redesign, I made the product name and brief description most prominent, and after that, an easy way to switch between views of different product models. Another prominent part of the product display page are images of the product. Features can be described in the text to the right of the product images, or reviewed with more detail further down the page. In the redesign, the steps for ordering online now are:

  1. Click “Products” in the top navigation, or hover over “Products” and select desired product
  2. Learn about product and its models on a single page
  3. Click “Add to Cart”
  4. Complete order


What’s next?


A low-fidelity prototype would ensure that the navigation of the website actually makes sense; users would have to be able to know how to complete certain tasks, such as Find out what exactly you get when you order a CompuTrainer, and with real copy provided, we could ensure that the right kind of information is shown on the right pages. (For example, do people want to know more about certain complex features? Less?) User testing would also target current users who use the website as a hub for Racermate-related news, and ensure that they aren’t alienated.
